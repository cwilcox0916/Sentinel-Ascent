# Currencies And Economy

This document lists every currency in `Sentinel Ascent`, how each is earned and spent, and — critically — **how every microtransaction pathway from the source genre is replaced with in-game progression**. There are no real-money purchases, no ads, no paid currency packs.

## 1. Economy Overview

`Verified` (genre); `New` (no-monetization replacements)

Sentinel Ascent uses many currencies because each long-term system needs its own pacing and sink pressure.

Buckets:

- In-run temporary currency (**Scrip**)
- Permanent core currency (**Alloy**)
- Premium-feel currency, freely earned (**Prisms**)
- Mode-specific currencies (**Cores**, **Insignia**, **Catalyst**)
- Order currencies (**Order Marks**, **Subnodes**)
- Equipment currencies (**Augment Fragments**, **Flux Crystals**)
- Tech-tree currency (**Cipher Keys**)

## 2. Scrip — In-Run Currency

`Verified` (renamed Cash)

| Field | Value |
|---|---|
| Role | Temporary run currency |
| Earned from | Enemy kills, Cycle bonuses, Resonance Surge windows, Interest/Cycle stat |
| Spent on | In-run Forge purchases (Attack/Defense/Utility tabs), Repair Kit refills |
| Persistence | **Resets at run end** |

Scrip is the only currency consumed by Auto-Procurement.

## 3. Alloy — Main Permanent Currency

`Verified` (renamed Coins)

| Field | Value |
|---|---|
| Role | Main permanent progression currency |
| Earned from | Enemy kills (modified by Alloy/Kill Bonus and Alloy Bonus), Stratum Milestones |
| Spent on | Forge upgrades, Research Bay project costs, Augment rank-ups |
| Persistence | Persistent |

Alloy is the forever-useful sink. It always has somewhere to go.

## 4. Prisms — Premium-Feel Currency

`Verified` (renamed Gems); `New` (all paid sources removed; daily/weekly free sources expanded)

| Field | Value |
|---|---|
| Role | Premium-feel currency, always earned through play |
| Persistence | Persistent |

### Earn paths (all free)

- **Floating Prisms** in-run — see section 14 (cap removed)
- **Daily Drops** — claim 30 Prisms/day, scaling streak bonus to 100/day at 7 consecutive days
- **Weekly Drop** — additional 50 Prisms claim
- **Achievement Vault** — 200+ achievements granting 5–500 Prisms each (replaces "offer walls")
- **Weekly Trial placement** — significant Prism payouts on the top-1000 reward table
- **Insignia exchange** — convert event currency to Prisms in the event shop
- **Daily Tasks** + **Weekly Tasks** — recurring Prism payouts
- **Order weekly chests** — Prisms on every contribution tier
- **Stratum Milestones** — guaranteed Prism payout at every 50-Cycle marker per Stratum
- **Behemoth drops** — small Prism chance on every Behemoth kill

### Spend paths

- Buying **Protocols** (10 Prisms each — halved from the source-pack 20 because there's no IAP top-up to balance against)
- Expanding **Research Slots**
- Buying **Augments** in the Augment Bay
- Entering **Weekly Trials** (free unlimited entries — the "Tournament Tickets" sink in the source genre is removed)

> **Prism slot expansion is gone.** The source pack's Card Slots cost up to 1000 premium currency per slot. In Sentinel Ascent, Protocol Slots are unlocked through the **Archive** tech tree (Cipher Keys) instead. See [04-progression-systems.md](04-progression-systems.md) section 6.

## 5. Cores — Arsenal Currency

`Verified` (renamed Power Stones)

| Field | Value |
|---|---|

`Inferred`

The build should distinguish in UI:

- **Augment Fragments** — rank up Augments

## 14. Floating Prisms (In-Run Pickups)

`New` (cap removed; rate scales with Cycle)

In the source pack, premium currency floating around the playfield is hard-capped (10 per run, ~once every 15 minutes, ~every 400–500 waves) — a deliberate friction against free-to-play income.

In Sentinel Ascent there is **no per-run cap**:

- Floating Prisms spawn periodically based on Cycle count
- Spawn rate ramps as Cycles climb (more Prisms in deeper runs)
- Each pickup grants 2 Prisms at low Cycles, scaling to 5 Prisms at deep Cycles
- Pickups disappear after 30 seconds if not collected

## 15. Microtransaction Replacement Table

`New` (every IAP/ad source from the source pack has an entry)

| Source-pack monetization | Sentinel Ascent replacement |
|---|---|
| Ad-watch gems (~5 Gems per ad) | **Daily Drops** — 30 Prisms/day claim, scaling to 100/day at a 7-day streak |
| Offer walls / TapJoy | **Achievement Vault** — 200+ achievements awarding 5–500 Prisms each |
| Real-money Prism packs | Removed entirely. Baseline run Prism rate increased ~4× to compensate |
| Monthly Core packs | **Stratum Milestones** — guaranteed Core + Prism payouts every 50 Cycles in each Stratum |
| Lab rush via Gems | **Overclock** project line — passive permanent research-speed multiplier (no paid skip) |
| Card (Protocol) Gem cost | Halved from 20 → 10 Prisms; Protocol copies also drop from Behemoths |
| Card-slot Gem walls (50 → 1000) | Slots moved into the **Archive** tech tree (Cipher Keys) |
| Module (Augment) Gem cost | Augments earned from Stratum challenge completions and Behemoth/fleet drops only |
| Floating-gem cap (10/run, 15-min cooldown) | **Cap removed**; spawn rate scales with Cycle |
| Tournament Tickets (Gems) | **Weekly Trials are free with unlimited retries** |
| Guild creation (50 Gems) | **The Order is single-player and free to join** |
| Premium themes (Tokens) | All Themes earned via Order Marks, Achievements, or Milestones |

## 16. Economy Loop Summary

`Inferred`

- Farm runs for Alloy and progression
- Spend Alloy on Forge and Research Bay
- Use Prisms to widen optional power systems
- Use Cores to deepen Arsenals
- Use Insignia in events for targeted purchases
- Use Order participation for Order Marks → Subnodes → Warden depth
- Use Weekly Trial reach for Cipher Keys → Archive tech tree

The principle preserved from the source pack: each currency feeds a different motivational loop, so the player is always pursuing several time horizons at once.

## 17. Reward Cadence Design

`Observed` + `Inferred`

Cadences are intentionally staggered:

- Seconds → minutes: Scrip, Alloy, in-run pickups
- Hours: Daily Tasks, Research Bay project progress
- Days: Weekly Tasks, event progress, Daily Drop streaks
- Weeks: Weekly Trial cycles, Order seasonal objectives, Heirloom accumulation

## 18. Clone Economy Principles

`Inferred` (preserved from source pack)

- Keep **Scrip** temporary; everything else persistent
- Let **Alloy** be the forever-useful sink
- Make **Prisms** flexible and valuable, but always reachable through play
- Keep **Cores** exclusive to Arsenals
- Use mode-specific currencies (**Insignia**, **Catalyst**, **Cipher Keys**) to gate mode-specific systems
- Never collapse currencies — each one represents a distinct motivational loop

## 19. Calibration Targets

`New` (no-monetization tuning bar)

A player who plays 30 minutes/day, never spends money, and never watches an ad should:

- Earn ~150 Prisms/day from Daily Drop + run + Achievements
- Acquire one new Protocol every 1–2 days early game
- Complete Auto-Procurement Tier 1 research within the first week of play
- Reach Stratum 3 (Order/Warden unlock) within the first two weeks
- Unlock fleet content (Stratum 14) within ~3 months of regular play

These are calibration targets, not hard guarantees. The economy build pass should validate against these.
